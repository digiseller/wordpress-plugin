Easy integration of http://digiseller.ru into you website.

== Installation ==

1. Upload `digiseller` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3.a. Configure from 'Settins->Digiseller' menu in WordPress
3.b. Use short code `[ds id="xxx" op="logo,search,cart,purchases,lang,cath,menu"]` in your conent
